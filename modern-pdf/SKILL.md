### 字体排版系统

**7 级字号体系**:
- Display (96pt) - 展示级，超大标题
- Hero (72pt) - 封面主标题
- Title (48pt) - 页面标题
- Heading (36pt) - 章节标题
- Body (18pt) - 正文
- Caption (14pt) - 说明文字
- Small (12pt) - 小字

**字体配对系统**:
- Classic Professional - 衬线 + 无衬线
- Modern Editorial - 展示 + 正文
- Minimal Clean - 纯无衬线
- Chinese English Mixed - 中英混排优化
- Technical Mono - 等宽字体技术文档
- Artistic Serif - 艺术衬线风格

## 快速开始

```javascript
const { ModernPDFUltimate, generateAllTemplates } = require('./scripts/modern-pdf-ultimate');

// 创建指定风格的生成器
const gen = new ModernPDFUltimate('academic');

// 封面页
gen.cover('论文标题', '答辩人：张三', '指导教师：李教授');

// 目录页
gen.toc([
    { title: '研究背景', page: 2 },
    { title: '文献综述', page: 3 },
    { title: '研究方法', page: 5 }
]);

// 学术风格专属页面
gen.outline('汇报提纲', [...]);
gen.literature('文献综述', [...]);
gen.methodology('研究方法', [...]);
gen.comparison('方法对比', [...]);
gen.conclusion('结论与展望', [...]);

// 结束页
gen.end('谢谢聆听', '敬请批评指正', 'email@university.edu');

gen.save('thesis-defense.pdf');
```

## 完整示例：学术报告

```javascript
const { ModernPDFUltimate } = require('./scripts/modern-pdf-ultimate');

const gen = new ModernPDFUltimate('academic');

// 封面
gen.cover('基于深度学习的图像识别研究', '答辩人：张三 | 导师：李教授', '计算机科学与技术学院');

// 目录
gen.toc([
    { num: '01', title: '研究背景与意义' },
    { num: '02', title: '文献综述' },
    { num: '03', title: '研究方法' },
    { num: '04', title: '实验设计与结果' },
    { num: '05', title: '结论与展望' }
]);

// 正文页
gen.heading('研究背景');
gen.body('本研究聚焦于深度学习在图像识别领域的应用...');
gen.infoCards('研究意义', [
    { icon: '理论', title: '理论意义', desc: '填补研究空白' },
    { icon: '应用', title: '应用价值', desc: '解决实际问题' }
]);

// 数据页
gen.dataStats('实验结果', [
    { value: '96.5%', label: '准确率', change: '+3.2%' },
    { value: '0.89', label: 'F1分数' }
]);

// 表格页
gen.table('方法对比', [
    { method: 'ResNet-50', accuracy: '92.1%', params: '25.6M' },
    { method: 'Ours', accuracy: '96.5%', params: '32M' }
]);

// 结论
gen.conclusion('结论与展望',
    ['提出新的图像识别方法', '取得SOTA结果'],
    ['优化模型结构', '扩展应用场景']
);

gen.end('谢谢聆听', '敬请各位老师批评指正');

gen.save('thesis-defense.pdf');
```

## 技术栈

- **Node.js + pdfkit** - 矢量 PDF 生成引擎
- **渐变支持** - 线性、径向、Mesh 渐变
- **字体嵌入** - 支持中英文字体
- **图片支持** - PNG/JPEG/SVG 嵌入
- **精确布局** - 像素级定位控制
