# Typography Guide - 字体排版指南

## 7 级字号体系

Modern PDF Ultimate 采用现代设计的 7 级字号体系，确保文档具有清晰的视觉层级。

### 字号定义

| 级别 | 名称 | 字号 (pt) | 用途 |
|------|------|----------|------|
| 1 | Display | 48 | 展示级标题、封面主标题 |
| 2 | Hero | 36 | 封面主标题、重要章节标题 |
| 3 | Title | 28 | 页面标题 |
| 4 | Section | 22 | 章节标题、小节标题 |
| 5 | Heading | 18 | 小标题、卡片标题 |
| 6 | Body | 12 | 正文内容 |
| 7 | Caption | 10 | 说明文字、注释、图表标注 |

### 字号使用原则

1. **层级清晰**: 每个级别都有明确的用途，避免混淆
2. **对比明显**: 相邻级别有足够的对比度
3. **适度使用**: 不在同一页面使用过多级别

### 行高设置

```javascript
const lineHeight = {
    display: 1.1,   // 超大标题需要紧凑的行高
    hero: 1.2,
    title: 1.3,
    section: 1.4,
    heading: 1.5,
    body: 1.6,      // 正文需要舒适的阅读行高
    caption: 1.5
};
```

## 字体配对系统

### 6 种字体配对风格

| 配对名称 | 标题字体 | 正文字体 | 适用场景 |
|----------|----------|----------|----------|
| Classic Professional | Helvetica-Bold | Helvetica | 商务文档 |
| Modern Editorial | Helvetica-Bold | Helvetica | 杂志报告 |
| Minimal Clean | Helvetica-Bold | Helvetica-Light | 设计作品集 |
| Artistic Serif | Times-Bold | Times-Roman | 文化主题 |
| Technical Mono | Courier-Bold | Courier | 技术文档 |
| Chinese English Mixed | Helvetica-Bold | Helvetica | 中英混排 |

### 字体选择建议

- **商务文档**: 使用 Classic Professional，专业可靠
- **创意设计**: 使用 Modern Editorial，现代时尚
- **艺术作品**: 使用 Artistic Serif，典雅优美
- **技术文档**: 使用 Technical Mono，清晰专业

## 中英文混排

### 原则

1. 中文正文使用 12pt，英文正文可使用 11pt
2. 中文标题字号比英文标题略大 2pt
3. 中文行高设置为 1.7，比英文略大

### 排版技巧

- 中文段落首行缩进 2 个字符
- 中英文混排时，保持一致的视觉节奏
- 标点符号使用全角或半角需统一

## 间距系统

### 8pt 网格系统

```javascript
const spacing = {
    xs: 4,    // 极小间距
    sm: 8,    // 小间距
    md: 12,   // 中等间距
    lg: 16,   // 大间距
    xl: 24,   // 超大间距
    xxl: 32,  // 巨大间距
    xxxl: 48  // 超巨大间距
};
```

### 使用场景

- xs: 图标与文字间距
- sm: 同组元素间距
- md: 不同组元素间距
- lg: 段落间距
- xl: 章节间距
- xxl: 页面区块间距
- xxxl: 页面顶部间距

## 文本对齐

### 对齐方式选择

| 对齐方式 | 适用场景 | 注意事项 |
|----------|----------|----------|
| 左对齐 | 正文、列表 | 默认选择，最自然 |
| 居中对齐 | 标题、封面 | 仅用于短文本 |
| 右对齐 | 数值、日期 | 数据表格中常用 |
| 两端对齐 | 长段落 | 注意避免河流现象 |

### 标题对齐建议

- 页面标题：居中对齐或左对齐
- 章节标题：左对齐
- 图表标题：居中对齐
- 表格标题：左对齐

## 文本样式

### 加粗使用

- 仅用于标题和强调内容
- 避免大段加粗文本
- 正文强调使用斜体而非加粗

### 斜体使用

- 引用内容
- 强调词句
- 外文词汇
- 图表注释

### 颜色使用

- 标题：使用主色调
- 正文：使用深色（#000000 - #1F2937）
- 说明文字：使用次要色（#6B7280 - #9CA3AF）
- 强调文字：使用强调色
